记账最大的敌人不是功能少，而是操作繁琐。

EnMoney 的所有交互都围绕「快速完成一笔记录」展开。

## 录入流程

1. 点击首页底部的「记账」按钮
2. 使用金额键盘输入金额
3. 选择或确认分类
4. 点击「保存」

完成。仅需几秒钟。

## 金额键盘

EnMoney 的金额键盘布局经过精心设计：

- 第一行：日期 / AC / ← / ÷
- 第二行：123
- 第三行：456
- 第四行：789
- 第五行：再记 / 0 / . / 完成

常用操作触手可及，减少手指移动距离。

## 可选信息

以下信息在需要时才填写，不会打断你的输入流程：

- **备注** — 添加文字说明
- **标签** — 为交易打上自定义标签
- **图片** — 拍摄或上传小票照片
- **账户** — 选择交易使用的账户（默认使用上次选择的账户）

## 再记功能

完成一笔记录后，点击「再记」按钮可以快速开始录入下一笔相似交易，分类和账户信息会保留上次的选择。

## 交易类型

- **支出** — 日常消费、购物、缴费等
- **收入** — 工资、红包、投资收益等
- **转账** — 账户之间的资金转移
- **余额调整** — 修正账户实际余额与记录之间的差异
