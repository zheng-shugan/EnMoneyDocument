第一次打开 EnMoney，你不需要进行任何复杂配置就能立即开始记账。

## 开箱即用

EnMoney 提供合理的默认设置，让你可以立即开始使用：

- **默认账本** — 一个主账本，记录所有日常交易
- **默认账户** — 现金、银行卡、微信支付、支付宝等常见账户类型
- **默认分类** — 餐饮、交通、购物、居住等日常分类
- **默认资产结构** — 清晰展示你的资产分布

随着使用深入，你可以逐步调整属于自己的管理方式。

## 核心概念

EnMoney 围绕几个核心概念展开：

| 概念 | 说明 |
|------|------|
| **交易** | 每一笔收入或支出记录，是 EnMoney 最基本的数据单元 |
| **账户** | 代表你的真实资产，交易本质上就是账户余额的变化 |
| **分类** | 交易的一个属性，帮助你了解钱花到了哪里 |
| **账本** | 所有交易的容器，默认一个主账本即可满足日常需求 |
| **计划** | 存钱目标，帮助你积累财富 |

## 从一笔交易开始

打开 App 后，点击首页底部的「记账」按钮即可进入记录页面。

只需要输入**金额**和**分类**（分类已提供默认选项），即可完成一笔记录。

其余信息（备注、标签、图片等）均为可选内容，需要时才填写。

> 最好的功能，是用户几乎感觉不到它的存在。
